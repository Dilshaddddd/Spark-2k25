Notes

SPARK is a welcoming program by the Open Source Software Club for first-year students, kickstarting your tech journey. No experience needed—just curiosity and a desire to learn!

Learn essential tools like HTML, CSS, and JavaScript to build websites. You'll write code, create web pages, and use Git to manage your projects.

Classes will be on campus, with seniors available for questions. Join our fun competitions to practice, learn, and connect with the tech community.

FAQS

What is Spark?

SPARK is a beginner-focused web development program designed for first-year students. It helps you learn everything from the ground up — starting with the basics of HTML, CSS, and JavaScript — and guides you through building real-world projects

Do I need any prior knowledge to attend SPARK classes?

No, SPARK is beginner-friendly and designed for complete newcomers. We start from scratch and help you level up step by step.

Will I get any practical experience or just theory?

SPARK is completely hands-on with live coding in every session.You'll build real-world projects and apply what you learn instantly.

Is SPARK only for CS/IT students?

No, SPARK is open to all First Year (FE) students, regardless of their branch.We believe anyone can code and create, no matter their stream.

What will I be able to do after the classes are completed?

You'll be able to develop and deploy a complete, fully functional website. This includes designing, coding, and hosting it online using GitHub.

How long is the SPARK program?

SPARK runs over several weekends, ensuring you have enough time to learn, practice, and complete small projects at your pace.

Is there any registration fee?

No, SPARK is completely free. It’s an initiative by the OSS Club to help students get started with web development.

What tools or software do I need?

You’ll just need a laptop and an internet connection. We’ll guide you through installing VS Code and other necessary tools in the first session.

What if I miss a session?

No worries! Notes and materials will be shared after each class. You can also reach out to the mentors for doubts.

Will mentors be available outside class hours?

Yes, our SPARK mentors will be available on WhatsApp groups to support you throughout the program.

How can I ask doubts during the session?

Sessions are interactive. You can ask your doubts live or drop them in the chat — our mentors will respond right away.

What topics will be covered in SPARK?

You’ll learn HTML, CSS, JavaScript, GitHub basics, building websites, deploying them online, and more.

Are there any competitions or quizzes in SPARK?

Yes! After each weekend, we conduct fun quizzes and small contests with goodies for the top scorers

What if I have zero coding experience?

That’s exactly who SPARK is made for! We start with the basics and guide you through everything step by step.

Can I use my phone instead of a laptop?

A laptop is highly recommended for hands-on practice. Using a phone will limit your ability to code effective

Will there be any guest speakers or alumni sessions?

Yes, we occasionally invite seniors and alumni to share their journeys and insights into the tech industry.

How can I stay updated about SPARK announcements?

Join the official SPARK WhatsApp group where all updates, materials, and announcements are shared.

What is a hackathon?

A hackathon is a fun coding event where you team up with others to build a cool tech project in a limited time — like a website, app, or tool.

Do I need to be an expert to participate in a hackathon?

Not at all! Beginners are welcome. Hackathons are more about learning, building, and collaborating than about being perfect.

Can I join a hackathon even if I only know basic HTML/CSS?

Yes! Many hackathon teams need designers, idea contributors, and basic developers. You’ll learn a lot during the event itself..
